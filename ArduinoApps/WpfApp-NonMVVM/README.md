## 更新 (2021/11/6)
リアルタイムにデータを記録できるアプリ CrayfishMonitor です。
 
 - Arduinoからのデータ受信
 - データの保存 (CSV形式)
 - リアルタイムなグラフ描画
 - グラフ画像の保存

が行えます。

モニタリングシステムとしてのリリース版です。このアプリを動かすにはランタイムパッケージ [.NET Core 3.1](https://dotnet.microsoft.com/download/dotnet/3.1) が必要です。Linux や Mac でも動かせるはずですが、まだ確認はできていません。
